# Sprouts

## :bullettrain_side: Tech Stack
Frontend: HTML, CSS, JQuery
Backend: Node.js, Express
AWS: EC2, Dynamo, Cognito

## :package: Setup
Node version: 9.2.0
NPM version: 5.5.1

### Install NPM packages

```bash
$> npm install
```

### Run app locally

```bash
$> npm start
```
